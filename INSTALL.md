# Установка BarsikCMD 🐾

BarsikCMD — лёгкая и минималистичная командная строка для Linux.  
Этот документ описывает самый простой способ установки.

---

## 📦 Установка через .deb пакет (рекомендуется)

BarsikCMD распространяется в виде `.deb` пакета, который подходит для:

- Linux Mint 22.3 (рекомендуемая система)
- Ubuntu 22.04+
- Других Debian‑based систем

### 1. Скачайте последний релиз

Перейдите в раздел Releases:

https://github.com/Barsik0396/BarsikCMD/releases

Скачайте файл вида:

```
barsikcmd_YYYY.MM.PATCH_amd64.deb
```

### 2. Установите пакет

Просто **откройте скачанный файл** и нажмите кнопку:

```
Установить пакет
```

Linux Mint автоматически выполнит установку и подтянет зависимости при необходимости.

---

## 🔄 Обновление BarsikCMD

BarsikCMD имеет встроенную команду обновления:

```bash
update
```

Она:

- проверяет последнюю доступную версию,
- скачивает обновление,
- устанавливает его,
- удаляет временные файлы.

Если обновления нет — BarsikCMD честно сообщит об этом.

---

## 🧹 Удаление BarsikCMD

Если нужно удалить программу:

```bash
sudo apt remove barsikcmd
```

BarsikCMD **не создаёт конфигов** и не оставляет скрытых файлов в системе.

---

## 📝 Примечания

- BarsikCMD работает только на Linux.
- Рекомендуемая система: **Linux Mint 22.3**.
