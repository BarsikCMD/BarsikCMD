# Installation 🐾

BarsikCMD распространяется в виде `.deb` пакета для Debian-based Linux.

---

## Требования

- Debian-based Linux (рекомендуется **Linux Mint 22.3** или **Ubuntu 22.04+**)
- Архитектура: `amd64`

---

## Установка через .deb (рекомендуется)

**1. Скачайте последний релиз**

Перейдите в [Releases](https://github.com/Barsik0396/BarsikCMD/releases) и скачайте файл вида:

```
barsik-cmd_YYYY.MM.PATCH_amd64.deb
```

**2. Установите пакет**

Откройте скачанный файл и нажмите **«Установить пакет»** — Linux Mint автоматически выполнит установку.

Или через терминал:

```bash
sudo dpkg -i barsik-cmd_*.deb
```

**3. Запустите**

```bash
barsikcmd
```

---

## Обновление

BarsikCMD обновляется встроенной командой:

```bash
update
```

Она проверяет последнюю версию, скачивает и устанавливает обновление, затем удаляет временные файлы. Если обновлений нет — сообщит об этом.

---

## Удаление

```bash
sudo apt remove barsik-cmd
```

BarsikCMD не создаёт конфигов и не оставляет скрытых файлов в системе.

---

## Возможные проблемы

**«Более поздняя версия уже установлена»**  
У вас стоит более новая версия. Если нужно переустановить:
```bash
sudo apt remove barsik-cmd
```

**`update` возвращает ошибку curl**  
Проверьте интернет-соединение и не прерывайте загрузку через Ctrl+C.
