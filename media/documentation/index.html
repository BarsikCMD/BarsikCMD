BarsikCMD

Кастомная командная строка для Linux. Быстрая, минималистичная, твоя.

barsik — ~
BarsikCMD> ver
BarsikCMD версия 2026.3.11.1
BarsikCMD> echo Привет, Linux!
Привет, Linux!
BarsikCMD>
Справочник команд
clear builtin Очистить экран ▾

Очищает содержимое терминала. Аналог системного clear, но встроенный прямо в BarsikCMD.

clear
Вывод
BarsikCMD> clear // вывод отсутствует — экран очищается
echo I/O Вывод текста ▾

Выводит переданный текст в стандартный вывод. Всё, что написано после echo, будет напечатано как есть.

echo [текст]
АргументОписание
[текст]Произвольная строка для вывода. Пробелы сохраняются.
Вывод
BarsikCMD> echo Привет, мир! Привет, мир!
BarsikCMD> echo  
exit system Завершить сессию ▾

Завершает текущий сеанс BarsikCMD и закрывает приложение.

exit
Вывод
BarsikCMD> exit Пока!
help info Список команд ▾

Выводит список всех доступных команд BarsikCMD.

help
Вывод
BarsikCMD> help Доступные команды:
  clear
  echo
  exit
  help
  ver
ver info Версия BarsikCMD ▾

Выводит текущую версию BarsikCMD.

ver
Вывод
BarsikCMD> ver BarsikCMD версия 2026.3.11.1
