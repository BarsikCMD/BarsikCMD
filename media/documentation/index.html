BarsikCMD

Кастомная командная строка для Linux. Быстрая, минималистичная, твоя.

barsik — ~
BarsikCMD v2026.3.4 — введи 'help' для списка команд
Доступно обновление: v2026.4.1. Введи 'update' для установки.
BarsikCMD> ver
BarsikCMD версия 2026.3.4
BarsikCMD>
Справочник команд
clear builtin Очистить экран ▾

Очищает содержимое терминала.

clear
Вывод
BarsikCMD> clear // вывод отсутствует — экран очищается
echo I/O Вывод текста ▾

Выводит переданный текст белым цветом.

echo [текст]
АргументОписание
[текст]Произвольная строка. Пробелы сохраняются.
Вывод
BarsikCMD> echo Привет, мир! Привет, мир!
exit system Завершить сессию ▾

Завершает текущий сеанс BarsikCMD.

exit
Вывод
BarsikCMD> exit Пока!
help info Список команд ▾

Выводит список всех доступных команд BarsikCMD.

help
Вывод
BarsikCMD> help BarsikCMD — доступные команды:
─────────────────────────────────
  clear — очистить экран
  echo — вывести текст на экран
  exit — выйти из BarsikCMD
  help — показать список команд
  info — информация о проекте. Аргументы: links
  meow — показать котика
  reload — перезапустить BarsikCMD
  update — проверить и установить обновления
  ver — показать версию BarsikCMD
─────────────────────────────────
info info Информация о проекте ▾

Выводит информацию о проекте. Без аргументов показывает список доступных подкоманд.

info [подкоманда]
ПодкомандаОписание
linksСсылки на GitHub и релизы проекта
Вывод — без аргументов
BarsikCMD> info Использование: info <аргумент>
Доступные аргументы:
  links — ссылки проекта
Вывод — info links
BarsikCMD> info links Ссылки проекта BarsikCMD:
─────────────────────────────────────────
  GitHub: https://github.com/barsik0396/BarsikCMD
  Релизы: https://github.com/barsik0396/BarsikCMD/releases
─────────────────────────────────────────
meow builtin Показать котика ▾

Выводит случайного ASCII-котика. Каждый раз показывается новый — подряд один и тот же не повторяется. Всего 5 вариантов.

meow
Вывод (один из вариантов)
BarsikCMD> meow /\_/\
( o.o )
> ^ <
reload system Перезапустить BarsikCMD ▾

Перезапускает BarsikCMD через execv — текущий процесс заменяется новым.

reload
Вывод — успешно
BarsikCMD> reload Перезапуск BarsikCMD...
Вывод — ошибка
BarsikCMD> reload Перезапуск BarsikCMD...
Ошибка: не удалось перезапустить BarsikCMD.
update system Проверить обновления ▾

Проверяет наличие новой версии и устанавливает её. Требует интернет-соединения и пароля для установки пакета.

update
Вывод — обновление доступно
BarsikCMD> update Проверка обновлений...
Доступна новая версия: 2026.4.1
Скачиваю пакет...
Размер пакета: 21 КБ
Устанавливаю пакет (потребуется пароль)...
BarsikCMD успешно обновлён до версии 2026.4.1!
Вывод — обновлений нет
BarsikCMD> update Обновлений нет. У вас актуальная версия (2026.3.4).
Вывод — ошибка сети
BarsikCMD> update Проверка обновлений...
Ошибка: не удалось получить информацию об обновлениях.
Канал обновлений: preview. При запуске BarsikCMD автоматически проверяет обновления (отключается флагом -disable-version-check-on-init).
ver info Версия BarsikCMD ▾

Выводит текущую версию BarsikCMD.

ver
Вывод
BarsikCMD> ver BarsikCMD версия 2026.3.4
Аргументы запуска
-help info Список аргументов ▾

Выводит список всех доступных аргументов запуска и завершает работу.

barsikcmd -help
Вывод
$ barsikcmd -help BarsikCMD — доступные аргументы запуска:
──────────────────────────────────────────────────────
  -help — показать этот список
  -run <команда> — выполнить команду и закрыть
  -run2 <команда> — выполнить команду и остаться
  -disable-version-check-on-init — не проверять обновления при запуске
  -load-type-select — (заглушка, будет реализовано позже)
──────────────────────────────────────────────────────
-run system Выполнить команду и закрыть ▾

Выполняет указанную команду BarsikCMD и сразу завершает работу.

barsikcmd -run <команда>
Пример
$ barsikcmd -run ver BarsikCMD версия 2026.3.4
Примечание: нельзя комбинировать с другими флагами.
-run2 system Выполнить команду и остаться ▾

Выполняет указанную команду BarsikCMD и переходит в интерактивный режим.

barsikcmd -run2 <команда>
Пример
$ barsikcmd -run2 ver BarsikCMD версия 2026.3.4
BarsikCMD v2026.3.4 — введи 'help' для списка команд

BarsikCMD>
Примечание: нельзя комбинировать с другими флагами.
-disable-version-check-on-init info Отключить проверку обновлений ▾

Отключает автоматическую проверку обновлений при запуске BarsikCMD.

barsikcmd -disable-version-check-on-init
Пример
$ barsikcmd -disable-version-check-on-init BarsikCMD v2026.3.4 — введи 'help' для списка команд

BarsikCMD>
-load-type-select info Заглушка ▾

Аргумент принимается, но функциональность будет реализована в будущих версиях.

barsikcmd -load-type-select
Вывод
$ barsikcmd -load-type-select Аргумент принят.
BarsikCMD v2026.3.4 — введи 'help' для списка команд

BarsikCMD>
