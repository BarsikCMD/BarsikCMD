# 📘 **Гайд: Как собрать BarsikCMD из исходников (Linux)**

BarsikCMD — это **утилита для Linux**.  
Сборка и запуск на Windows PowerShell/CMD **не поддерживаются**.  
WSL **не рекомендуется**, так как команда `update` может работать нестабильно.

Рекомендуемая система: **Linux Mint**  
(но подойдёт любой Linux с предустановленным компилятором).

---

## 📦 Требования

На Linux Mint уже есть:

- **gcc / g++** — компилятор  
- **git** — система контроля версий  
- **терминал** — естественно  

Дополнительно требуется установить:

### **libcurl4-openssl-dev**  
Необходим для работы команды `update`.

Установка:

```bash
sudo apt install libcurl4-openssl-dev
```

---

## 📥 1. Клонирование репозитория

```bash
git clone https://github.com/barsik0396/BarsikCMD.git
cd BarsikCMD
```

---

## ⚙️ 2. Сборка проекта

BarsikCMD собирается через `make`, но сборка выполняется **внутри папки Linux‑версии**.

Перейдите в нужную директорию:

```bash
cd code/linux/mint
```

Запустите сборку:

```bash
make
```

После успешной сборки появится исполняемый файл `barsikcmd`.

---

## ▶️ 3. Запуск

```bash
./barsikcmd
```

---

## 🧪 4. Тесты после сборки

После сборки рекомендуется выполнить три теста, чтобы убедиться, что BarsikCMD работает корректно.

---

### **1. Проверка версии через run‑режим**

```bash
./barsikcmd -run ver
```

Ожидаемый результат:

- BarsikCMD запускается  
- выполняет команду `ver`  
- выводит текущую версию (например, `2026.3.3`)  
- завершает работу  

---

### **2. Проверка help в режиме run2**

```bash
./barsikcmd -run2 help
```

Ожидаемый результат:

- BarsikCMD запускается  
- выводит справку (`help`)  
- **не завершает работу**  
- остаётся в интерактивном режиме (`barsikcmd>`)  

---

### **3. Проверка команды update внутри BarsikCMD**

Запуск:

```bash
./barsikcmd
```

Внутри программы:

```
update
```

Ожидаемый результат — один из двух вариантов:

### **Вариант А — обновление доступно**
BarsikCMD:

- проверяет сервер  
- начинает скачивание  
- может запросить пароль через графический UI Linux Mint (стандартное окно авторизации)  
- успешно обновляется  

### **Вариант Б — обновлений нет**
BarsikCMD выводит сообщение вида:

```
Нет доступных обновлений
```

Ошибок быть **не должно**.

---

## 🎉 Готово!

BarsikCMD успешно собран и работает.  
Теперь можно использовать утилиту, тестировать команды и участвовать в развитии проекта.
