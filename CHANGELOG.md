# Changelog
Все заметные изменения в проекте **BarsikCMD** будут документироваться в этом файле.

Формат основан на стандарте **Keep a Changelog**  
и проект использует версионирование в формате **Год.Месяц.Патч** (например: 2026.3.3).

---

## [2026.3.3] — Update fix
### Исправлено
- Исправлено поведение команды `update`: BarsikCMD больше не считает, что доступно обновление, если его нет.
- Улучшена стабильность проверки версии.

---

## [2026.3.2] — First update
### Добавлено
- Команда `update` для автоматического обновления BarsikCMD.
- Цветной вывод в терминале.

### Изменено
- Улучшена читаемость вывода команд.

### Известные проблемы
- Команда `update` иногда ошибочно определяет наличие обновления (исправлено в 2026.3.3).

---

## [2026.3.1] — First release
### Добавлено
- Первый официальный релиз BarsikCMD.
- Базовые команды: `clear`, `echo`, `exit`, `help`, `ver`.
- Базовый интерфейс командной строки.
- Поддержка Linux Mint.

---

## Формат версий
BarsikCMD использует схему:
```
Год.Месяц.Патч
```
Пример: `2026.3.3` — март 2026, третий патч.
